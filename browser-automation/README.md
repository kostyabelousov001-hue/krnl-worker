# ⚡ KRNL Lightning Distributed Crawler v1.2
> **Готовый B2B SaaS-продукт и инструмент для генерации лидов «из коробки».** 

Мощный распределенный веб-краулер на базе Playwright, который собирает целевые B2B-контакты (название компании, точный рейтинг, количество отзывов, телефон, веб-сайт) с Google Maps в реальном времени, а затем автоматически переходит на сайты компаний для сбора электронных писем (emails) и социальных сетей.

Благодаря **распределенной архитектуре (Host-Worker)**, скрапинг можно масштабировать на десятки параллельных машин, кратно увеличивая скорость работы и обходя любые лимиты по IP.

## 💎 Уникальные торговые преимущества (УТП) для продажи

Если ты планируешь продавать этот скрипт как готовый софт или использовать его для оказания услуг на фрилансе (Fiverr, Upwork, Kwork), вот твои главные козыри:

1. **⚡ Распределенный скрапинг (Host-Worker):**
   Один клик на главной машине (Host) — и десятки подключенных воркеров (Workers) на других компьютерах или виртуальных серверах мгновенно разбирают очередь и парсят сайты параллельно. Обмен задачами идет по сверхбыстрому протоколу WebSockets.
2. **🎯 Обход защиты Google Maps (Lite Mode & UserAgents):**
   Google часто переводит автоматические запросы в режим "Lite Mode", скрывая количество отзывов и рейтинг. Наш краулер имитирует реальные действия пользователя с помощью stealth-заголовков, подмены viewport и постоянной сессии, гарантируя 100% точность сбора отзывов.
3. **📊 Выгрузка в 7 форматов (All-in-One):**
   * **`leads.xls` (Excel XML 2003):** Уникальный экспорт с жестко заданными типами данных. Рейтинг сохраняется как число. **Больше никаких лагов Excel**, превращающих рейтинг `4.8` в дату `08.апр`.
   * **`leads.html` (Premium Dashboard):** Интерактивная веб-страница в темном неоновом стиле с поиском по базе, карточками статистики и кнопками для быстрого скачивания других форматов.
   * **`leads.md`:** Красивая Markdown-таблица для Obsidian, Notion или GitHub.
   * **`leads.csv`:** Классический CSV с разделителем `;` и поддержкой кодировки UTF-8 BOM.
   * **`leads.json`:** Чистый JSON-массив объектов для интеграции с базами данных и CRM.
   * **`leads.xml`:** Валидный XML для корпоративных корпоративных систем.
   * **`leads.tsv`:** Текстовый файл с табуляцией.
4. **🔄 Умный инкрементальный кэш и дедупликация:**
   Скрипт сохраняет промежуточные результаты в `leads_temp.ndjson`. При повторном запуске он автоматически загружает старые лиды, отсекает дубликаты (сравнивая уникальные URL-адреса Google Maps, чтобы не потерять разные филиалы одной сети) и скрапит только новые контакты.
5. **🧼 Чистые данные без мусора:**
   Скрапер на лету вырезает скрытые иконки-глифы Google Maps (такие как `` или ``), которые ломают верстку при экспорте в обычных парсерах.

## 🚀 Быстрый старт

### Требования
* Node.js v18 или выше
* Установленные зависимости (Playwright, ws, csv-writer)

### Установка
1. Склонируй или скачай эту папку.
2. Открой терминал в папке `browser-automation` и выполни установку зависимостей:
   ```bash
   npm install
   ```
3. Установи Chromium-браузер для Playwright:
   ```bash
   npx playwright install chromium
   ```

### Варианты запуска

#### 1. Автоматический режим (в один клик)
Запуск скрапинга по заданному запросу и лимиту. Скрипт сам запустит локальный хост, соберет данные, выгрузит все 7 форматов и покажет красивую таблицу в конце:
```bash
node distributed-app.js --auto --query "real estate agents Dubai" --limit 50
```

#### 2. Интерактивный TUI-режим
Запусти программу без флагов:
```bash
node distributed-app.js
```
Откроется графический терминальный интерфейс (TUI):
* Нажми **`H`**, чтобы запустить эту машину в режиме **HOST**.
* Нажми **`Q`**, чтобы ввести поисковый запрос (например: `dentists New York`).
* Нажми **`L`**, чтобы ввести лимит сбора (например: `100`).
* Нажми **`S`**, чтобы начать молниеносный скрапинг!
* Нажми **`Esc`**, чтобы выйти из программы.

#### 3. Распределенный режим (Масштабирование)
* На **главном сервере** запусти `node distributed-app.js`, нажми `H` (Host).
* На **других машинах** запусти тот же скрипт, нажми **`W`** (Worker) и введи IP хоста (например: `ws://192.168.1.50:9090`).
* Воркеры подключатся к хосту, и как только хост начнет работу кнопкой `S`, они разделят задачи и будут парсить данные в N раз быстрее!

## 📁 Структура файлов проекта
* `distributed-app.js` — ядро приложения, TUI-интерфейс, WebSocket сервер/клиент, логика распределения батчей и экспортеры.
* `stealth-scraper.js` — модуль скрапинга страниц Google Maps и глубокого парсинга сайтов (поиск почт, телефонов и соцсетей с обходом внутренних ссылок).
* `leads_temp.ndjson` — кэш-файл с промежуточными результатами для возможности продолжения работы.
* `leads.html` — премиальный интерактивный дашборд.
* `leads.xls` — валидная таблица Excel без бага с датами.
* `leads.csv` / `leads.json` / `leads.md` / `leads.xml` / `leads.tsv` — остальные форматы экспорта.
