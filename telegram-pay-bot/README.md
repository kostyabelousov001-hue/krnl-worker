# Telegram TON Monetization Bot

This is a complete Node.js Telegram bot that accepts payments in TON directly to a specified wallet address. It monitors incoming blockchain transactions for a specific amount and unique comment (memo) to automatically activate premium privileges for users.

## Features

- `/start` - Greeting and system explanation.
- `/pay` - Generates payment details, shows target address, amount, unique comment, and builds a direct pay link for Tonkeeper.
- `/check` - Scans the blockchain using TON Web API to verify the transaction.
- `/premium` - Restricted command only accessible to users who completed the payment.
- Stores user statuses in a local `users.json` file.

## Setup and Running

1. Install dependencies in `telegram-pay-bot/`:
   ```bash
   npm install
   ```
2. Configure `.env` in the root folder with:
   - `TELEGRAM_BOT_TOKEN`: Token from [@BotFather](https://t.me/BotFather).
   - `TON_WALLET_ADDRESS`: Your TON address where payments should be sent (you can use the address generated by `node generate-wallet.js`).
   - `PREMIUM_PRICE_TON`: Premium access price (defaults to `0.5` TON).
   - `TONCENTER_API_KEY` (optional): To avoid rate limits on the Toncenter provider.
3. Start the bot:
   ```bash
   node index.js
   ```
